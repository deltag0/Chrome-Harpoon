Thank you for downloading Chrome-Harpoon!

Chrome-Harpoon is an extension similar to the Neovim plugin made by ThePrimeagen (check it out!), but you don't need to know any of that to start using Harpoon.
Harpoon is a minimalist extension that lets users mark/pin tabs (up to 10) and switch to them using keyboard shortcuts. It also includes shortcuts to switch back and forth between the last-used tabs easily.
It's also fully compatible with Vimium!

To start using Chrome-Harpoon, here's a breakdown of the commands:

  • Alt a will mark/pin a tab
  • Alt 0-9 will switch to the marked tab (if existing), with 1 being the first marked tab and 0 the last
  • Alt z will switch tabs to the previously selected tab
  • Alt y will undo a ctrl z switch

You can also click on the extension icon to remove a pinned tab at any time.
